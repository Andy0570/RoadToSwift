在本教程中，您将学习如何使用 PhotoKit 访问和修改照片、智能相册和用户收藏夹。您还将学习如何保存和恢复对照片所做的编辑。

要点：

* PhotoKit 的权限模型。
* 访问所有照片、智能相册和用户收藏相册。
* 从资产中获取图像。
* 修改资产元数据。
* 编辑资产的图像。
* 保存资产修改。
* 还原资产修改。
